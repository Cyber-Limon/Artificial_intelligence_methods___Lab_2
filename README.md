# Лабораторная работа 2 | Вариант 1

### Задание
#### Формирование равных по сложности экзаменационных билетов из вопросов
Имеется перечень вопросов, каждый из которых относится к определенному разделу дисциплины и имеет некоторую сложность. 
Требуется сформировать равные по сложности экзаменационные билеты, состоящие из 2-х вопросов, не относящихся к одному разделу дисциплины.

### Требования:
1.	Агентов создавать в коде программы.
2.	Исходные данные для агентов должны храниться в файле любого формата.
3.	Выходные данные – результат работы – представить в виде файла, удобного для прочтения.
4.	Реализовать МАС в виде сервера и клиентов. Обеспечить возможность запуска как минимум с 2 машин: на 1 машине – сервер и клиент, на 2 машине – клиент.
